Wise Platform Integration Guides

Welcome to the Wise Platform API Docs Guides page! Here, developers can access the tools and tutorials needed to use the Wise Platform API and quickly and easily integrate it into their applications.

Our guides provide step-by-step instructions on everything from getting started to advanced topics like customizing the API to your unique needs. Start exploring today and discover the power of the Wise Platform API!

If you are unsure of which guide you should be using for your integration, please contact us .

Connection Guides

Each Connection Guide focuses on how a partner connects to Wise and how they leverage the platform and products. If you are unsure of which connection type you will require, please speak with us .

Customer Accounts with Partner KYC
For Banks, Licensed Institutions, and those with trusted outsourced KYC partner status

Create and link to your customer's Wise accounts to create transfers, balances, and utilise other products.

Customer Accounts with Wise KYC
For Partners that require Wise to complete KYC

Create and link to your customer's Wise accounts to create transfers, balances, and utilise other products.

Customer Accounts with Prefilled Wise KYC
For Customers who want to review a series of pre-filling forms instead of filling them in from scratch.

Make onboarding new users easier by sending portion of required customer data upfront via API.

Partner Account
For Partners that want to use their own Wise account

Use your own business' Wise account to programmatically create transfers, create batch payments, and utilise other products.

International Receive

Allow your customers to receive funds via Swift with Wise as your correspondent.

Affiliates

After becoming an affiliate partner, use this guide to easily pull current and historical rates, as well as temporary quotes with fees and pricing to display in your customers experience.

Open Banking

Learn about accessing our sandbox and production environments, using client credentials to create and access user profiles, and leveraging OAuth 2.0 to connect to existing users.
