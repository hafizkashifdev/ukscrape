Wise Platform Docs

Welcome to the Wise Platform API Documentation! With our API, you can create smarter and more reliable payment experiences for your customers leveraging the power of Wise Platform.

Here you'll find comprehensive documentation and resources to help you build and integrate with the Wise Platform API. The API enables you to securely access and manage Wise accounts and data, design and customize your payment, cards, and Wise account flows, and leverage our payments and banking infrastructure.

Getting to know the Wise Platform API

Whether you're a developer, business owner, or financial institution, the Wise Platform API documentation provides the tools and resources needed to create innovative payment solutions. Documentation is organized into three key areas.

  • Generate your API Token
  • Select our API Sandbox environment

When you are ready to do this in Production, you can get started by creating a Wise Account:
